# DeliveryServiceTest

Для запуска откройте консоль nuget (Вверху VS Studio Кнопки: Файл, Правка, ВИД, Открываем Вид и переходим в другие окна, там есть
Консоль диспетчера пакетов). Вписываем туда две команды:

>Add-Migration InitialCreate
>Update-Database

У нас создаётся БД в MSSQL на локалке. После этого можно добавить тестовые данные в таблицу Order и открывать приложение через отладку,
Либо опубликовать и открыть так. 

Логи пишутся в нижнем боксе, и сохраняются в файл, который лежит в корне приложения 
(DeliveryServiceTest\bin\Debug\net8.0-windows\delivery_log.txt)
